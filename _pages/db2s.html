--- layout: page title: B2S Server permalink: /db2s/ --- **Source:** [GitHub](https://github.com/vpinball/b2s-backglass/releases)  ·  **Current version:** 2.1.2  ·  [B2S Wiki](https://github.com/vpinball/b2s-backglass/wiki) **Description:** Software used to display an image on the backglass in Visual Pinball X. --- ## What is B2S Server? The B2SServer allows a backglass image to be displayed in a Visual Pinball X table. The backglass file (`tablename.directb2s`) is an XML file with images and instructions on when to display certain smaller images that simulate lamps behind the backglass. The trigger can come either from a pinball ROM via VPinMAME or directly from commands in the table VB script. --- ## Setting It Up You might already have a version installed. To check your version, launch a table and right-click on the backglass — this will show the installed version. Version 2 has changed significantly in both look and setup. Download the latest from [GitHub](https://github.com/vpinball/b2s-backglass/releases). Rename the downloaded folder from the long name (e.g. `B2SServer-2.0.3-a1b7e6d-Release-win-x86-x64`) to simply **B2SServer**. --- ## Old Version Users If you have an existing B2SServer installed, back up all files that belong to the server — anything starting with `B2S`, `B2SBackglassServer.dll`, `B2SBackglassServerEXE.exe`, `B2SServerPluginInterface.dll`, etc. It is always a good idea to back up `ScreenRes.txt`, `B2STableSettings.xml`, and your plugins folder. Copy your `ScreenRes.txt` into the new B2SServer folder. You can place the B2SServer folder anywhere — a common choice is the root of your Visual Pinball folder. Delete all instances of B2SServer located in your `VisualPinball/Table` folder (this is where the old version was installed). In your tables folder, delete everything except the `table.xml` file. --- ## New Version Users Copy the B2SServer folder into your Visual Pinball folder and run `B2SBackglassServerRegisterApp.exe`. Accept the security warning — this registers the software as a COM object.
You will get a warning if it is already registered — you can still say YES. Run `B2S_ScreenResIdentifier.exe` to set up your screen layout. --- ## Screen Setup **1 Screen Setup** Select how many screens your setup has. You can then move the box to the appropriate screens and the Backglass and DMD will be positioned correctly when launching tables. **2 Screen Setup** With more than one screen, the software detects it and allows multiple screen options. Move the backglass screen to the 2nd monitor and check the full screen box. --- ## In-Game Backglass Settings [B2S Settings reference](https://github.com/vpinball/b2s-backglass/wiki/Settings) While a VPX table is loaded (not a Pup Pack), right-click on the backglass to access the settings menu. The most commonly used feature is enabling or disabling the Speaker Grill — use the dropdown to select Hide and save. You must reload the table for changes to take effect. --- ← [Back to Software](/software/)
