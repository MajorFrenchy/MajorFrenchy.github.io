A reference for common virtual pinball terms, acronyms, and jargon.

---

| Term | Definition |
|------|------------|
| **VPX** | Visual Pinball X — the main virtual pinball simulation engine |
| **VPW** | Visual Pinball Wheel — community resource for VPX tables |
| **B2S** | Back2Screen — software that drives the backglass display |
| **DMD** | Dot Matrix Display — the score/animation screen on real and virtual machines |
| **DOF** | DirectOutput Framework — connects software to physical feedback toys |
| **PinballX** | A popular frontend/launcher for virtual pinball cabinets |
| **PinUP Popper** | Another popular pinball frontend with media support |
| **Plunger** | The spring-loaded ball launcher, often replicated physically in cabs |
| **Nudge** | Physically tilting the cabinet to influence the ball — simulated via accelerometer |
| **ROM** | Read-Only Memory — the game logic file for WPC-era tables |
| **VPinMAME** | Emulator that runs pinball ROMs inside VPX |
| **SSF** | Surround Sound Feedback — positional audio for cabinet immersion |
| **Shaker** | Motor that vibrates the cabinet in sync with game events |
| **Solenoid** | Electromechanical actuator that creates tactile pops and knocker effects |
| **Contactors** | High-current solenoids for flipper feedback |
| **LedWiz / Pinscape** | Controller boards used to drive DOF output devices |
| **Batocera** | Linux-based retro gaming OS with VPX Standalone support |
| **VPX Standalone** | A cross-platform port of VPX that runs on Linux/Mac/iOS |
