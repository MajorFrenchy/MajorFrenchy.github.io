There is a very large community creating original tables and real machine recreations. **VPX Features:** - Simulates pinball table physics and renders the table with DirectX or OpenGL - Simple editor to (re-)create any kind of pinball table - Live editing in the rendered viewport - Table logic controlled via Visual Basic Script - Over 1050 real/unique pinball machines recreated, plus 550+ originals - Emulation of real pinball machines via VPinMAME - Configurable camera views for cabinet mode - Support for Tablet/Touch input and specialized pinball controllers - Support for Stereo3D, Head Tracking via BAM, and VR HMD rendering --- ## Downloads {#downloads} | Software | Link | Version | |----------|------|---------| | **VPX** | [GitHub](https://github.com/vpinball/vpinball/releases) | 10.8 RC6 | | **VPinMAME** | [GitHub](https://github.com/vpinball/pinmame/releases) | 3.6 | | **B2S Server** | [GitHub](https://github.com/vpinball/b2s-backglass/releases) | 2.1.2 | --- ## Installing Visual Pinball {#installation}
1. Download from GitHub
Download Visual Pinball X from the official GitHub. Download the full installer first before upgrading to 10.8.
2. Unblock the file
Right-click the installation file and select Properties.
3. Check Unblock
Check the Unblock box and click Apply.
4. Run the installer
Double-click to install and follow the prompts.
5. Choose install path
You can install it anywhere. A good practice is to create a folder called Vpinball and install all related software inside it.
6. Select components
Select B2S Backglass and .Net. If you have a real DMD, you can also select PinDMD.
7. Launch VPX
Double-click the VisualPinballX shortcut on your desktop. Go to Preferences → Video/Graphic options.
8. Configure video settings
Select your video card and monitor resolution. Uncheck Force Exclusive Fullscreen. Adjust the texture slider to reduce lag or memory errors.
9. Find tables via VPS
The Virtual Pinball Spreadsheet helps you find tables by keywords, along with hosting site, versions, ROM, Backglass, and media files.
10. Table hosts
Tables are hosted on VPUniverse, VPForums, and Pinball Nirvana. Location is decided by the table author.
11. Read the file details
Familiarize yourself with important details such as what version of software is required.
12. ROMs
Real table recreations require a ROM for emulation. Original tables do not need a ROM. VPS includes a link to find the ROM for each table.
13. Download the backglass
On the VPS site, backglasses are also listed. Download one and extract it to your VisualPinball/Table folder.
14. Rename the backglass
Rename the .directb2s file to match your table name exactly and move it to your VisualPinball/Table folder.
15. Launch VPX
Double-click the desktop shortcut, then go to File → Open and select a VPX table to open it in editor mode.
16. Editor mode
In editor mode you can view the script, launch the interactive view, or launch the table directly.
17. Launch the table
Press F5 or click the launch icon to start the table.
18. Play!
If you get an error, it's most likely due to the wrong version of software. Check that you meet the minimum requirements (VPX 10.8, VPinMAME 3.6).
--- ## Upgrading VPX {#upgrading-vpx}
1. Check your current version
Open VPX and go to Help → About.
2. Download the upgrade
You only need the full installer once. For upgrades, just drop in the new files. Always backup the old files first. Beta versions are on GitHub.
3. Pick the right build
Choose the Release-Win version. You can use the X86 version.
4. Unblock the file
Right-click → Properties → check Unblock → Apply.
5. Extract the zip
Extract the downloaded archive.
6. Copy and replace
Move the extracted files into your VisualPinball folder. When prompted to overwrite, say Yes. You're done!
--- ## Upgrading VPinMAME {#upgrading-vpinmame}
1. Check current version
Navigate to your VisualPinball/VPinMAME folder and double-click Setup.exe.
2. Note your version
The setup will display your currently installed version.
3. Download from GitHub
Go to the VPinMAME GitHub releases. Pick the Release-Win X86 version.
4. Unblock the file
Right-click → Properties → check Unblock → Apply.
5. Extract the zip
Extract the downloaded archive.
6. Copy and replace
If you have a VPMalias file with content, keep the old one. Copy and replace all other files into the VPinMAME folder.
7. Run Setup.exe
Open Setup.exe from the VPinMAME folder. The new version number should be highlighted. Click Install and follow the prompts.
8. Check paths
Click the Path button and verify your paths are set correctly.
